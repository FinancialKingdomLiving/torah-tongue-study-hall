Parsing means identifying what the word is doing grammatically.

Verb Parsing Formula

Stem + Form + Person + Gender + Number + Root + Meaning

Example: שָׁמַר = Qal perfect 3ms from שמר, “he kept/guarded.”

Noun Parsing Formula

Gender + Number + State + Root + Meaning
